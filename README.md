# BlockType

A C++26 compiler with bilingual support (Chinese/English) built with AI assistance.

## Features

- **C++26 Support**: Full support for C++26 standard features
- **Bilingual Programming**: Native support for Chinese and English keywords
- **AI Integration**: AI-assisted diagnostics and code suggestions

## Building

### Prerequisites

- CMake 3.20+
- C++23 compiler (Clang 18+, GCC 13+, or MSVC 19.35+)
- LLVM 18+

### Quick Start

```bash
./scripts/build.sh Release
./scripts/test.sh
```
